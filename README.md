# Coupa Request Challenge

An interactive training game for new joiners to learn how to correctly complete a Coupa supplier request.

## About

Part of the GSS onboarding curriculum (Week 2). Players review a mock Coupa request form, spot errors across six mandatory fields, and learn why each field matters for the approval workflow.

## Fields covered

- Justification
- Request Type (New, Renewal, Termination, Remediation, RFX)
- Start and End Date
- Dollar Amount
- Legal Framework
- Type of Agreement

## How to play

Visit the live URL and work through 8 scenarios. Each answer includes feedback explaining the error and its impact on the workflow.
